What is Paltform?

Paltform is a platform game online created by Valv0 rainbowdash and betawolfy. It is a platform game that is designed to be played on a computer. the principe is simple, if you fall, you're teleported to the beginning of the level, so you can retry. there is no life, no game over, only victory.

actual version : 1.0
